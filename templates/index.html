AI-Driven Environmental Monitoring
and Forecasting System

The objective of this project is to create an AI-based environmental monitoring and forecasting system with two core components. The first component is a forecasting module that predicts the Air Quality Index (AQI) based on key environmental factors such as PM2.5, NO2, CO, and others. This system will enable real-time predictions of air quality, helping users understand pollution levels and their potential impact on health. The second component is an image classification module that uses deep learning algorithms to classify satellite images into predefined categories: Cloudy, Desert Area, Green Area, Unknown, and Water Body
